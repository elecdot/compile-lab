# Utilities

Directory entry point for project-specific utility files.

Use this directory only for project-specific helper files that do not belong in
`src/` or `scripts/`.

## Scope

- Shared data files.
- Small development utilities.

## Local Conventions

- Keep this directory narrow; prefer `src/` for implementation code and
  `scripts/` for executable workflows.
- Document why each utility file exists.
- Do not use this directory as a miscellaneous dump for temporary files.
