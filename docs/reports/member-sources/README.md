# Member Sources

Group-member report and implementation source material for final report
integration.

## Files

- [lab1.md](lab1.md): Markdown conversion of `lab1.docx`, the group-member Lab 1
  lexical-analysis report manual.
- [lab2.md](lab2.md): Markdown conversion of `lab2.docx`, the group-member Lab 2
  recursive-descent parser report manual.
- `lab1-media/` and `lab2-media/`: image assets extracted from the corresponding
  `.docx` files by Pandoc.

## Usage Notes

- These are group-member-completed Lab 1 and Lab 2 report manuals; the Lab 2
  manual documents the recursive-descent implementation.
- Treat [lab1.md](lab1.md) and [lab2.md](lab2.md) as important factual sources
  for the group members' portions of the final report.
- Use these converted reports to verify member-owned implementation claims,
  grammar descriptions, extension descriptions, test cases, and screenshots.
- If either `.docx` source is regenerated, rerun the Markdown conversion from
  this directory and keep the corresponding media directory in sync.
